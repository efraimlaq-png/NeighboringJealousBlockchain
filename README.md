# Friday Bot

Um bot para Discord construído com Node.js, `discord.js` e `dotenv`.

## Sobre

Este repositório contém o código-fonte do bot Friday, incluindo as configurações e dependências necessárias.

## Instalação

```bash
npm install
```

## Uso

Crie um arquivo `.env` com suas credenciais do Discord e execute:

```bash
node friday.js
```

## Estrutura

- `friday.js` - entrada principal do bot
- `package.json` - dependências e scripts
- `.gitignore` - arquivos ignorados pelo Git
